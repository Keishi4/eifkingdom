Версия 1.2.0.1
 - Адаптирована для запуска на Windows системах (Win95/NT 3.5 и выше).
   Для запуска в Windows NT 3.5x (и возможно Win95) требуется библиотека
   'msvcrt.dll' минимум версии 4.0, также требуется поддержка русского
   языка для корректного отображения текста.

 - Исправления багов:
   * отсутствие запроса на ввод в одной из функций игры;
   * отсутствие вывода суммы затрат в одной из функций игры;
   * формат вывода при печати очков времени правления;
